3. To use the tool open cmd & write the following convention:
	[%JDK_PATH\bin\java% -jar FixMyFile.jar %PATH_TO_CORRUPTED_XML_FILE% %PATH_TO_FILE_WHERE_XML_IS_FIXED%]
